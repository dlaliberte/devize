Linear Gradient Examples

Basic Horizontal Gradient

A simple left-to-right linear gradient with two colors.

{
  type: "linearGradient",
  id: "basic-horizontal-gradient",
  x1: "0%",
  y1: "0%",
  x2: "100%",
  y2: "0%",
  stops: [
    { offset: "0%", color: "#3498db" },
    { offset: "100%", color: "#9b59b6" }
  ]
}

Vertical Gradient

A top-to-bottom linear gradient.

{
  type: "linearGradient",
  id: "vertical-gradient",
  x1: "0%",
  y1: "0%",
  x2: "0%",
  y2: "100%",
  stops: [
    { offset: "0%", color: "#2ecc71" },
    { offset: "100%", color: "#f1c40f" }
  ]
}

Diagonal Gradient

A diagonal linear gradient from top-left to bottom-right.

{
  type: "linearGradient",
  id: "diagonal-gradient",
  x1: "0%",
  y1: "0%",
  x2: "100%",
  y2: "100%",
  stops: [
    { offset: "0%", color: "#e74c3c" },
    { offset: "100%", color: "#3498db" }
  ]
}

Multi-Stop Gradient

A gradient with multiple color stops.

{
  type: "linearGradient",
  id: "multi-stop-gradient",
  x1: "0%",
  y1: "0%",
  x2: "100%",
  y2: "0%",
  stops: [
    { offset: "0%", color: "#1abc9c" },
    { offset: "25%", color: "#3498db" },
    { offset: "50%", color: "#9b59b6" },
    { offset: "75%", color: "#e74c3c" },
    { offset: "100%", color: "#f1c40f" }
  ]
}

Gradient with Opacity

A gradient with varying opacity levels.

{
  type: "linearGradient",
  id: "opacity-gradient",
  x1: "0%",
  y1: "0%",
  x2: "100%",
  y2: "0%",
  stops: [
    { offset: "0%", color: "#e74c3c", opacity: 1 },
    { offset: "50%", color: "#e74c3c", opacity: 0.5 },
    { offset: "100%", color: "#e74c3c", opacity: 0 }
  ]
}

Gradient with Spread Method

A gradient with repeat spread method.

{
  type: "linearGradient",
  id: "spread-method-gradient",
  x1: "0%",
  y1: "0%",
  x2: "25%",
  y2: "0%",
  spreadMethod: "repeat",
  stops: [
    { offset: "0%", color: "#3498db" },
    { offset: "100%", color: "#2ecc71" }
  ]
}
