Text Visualization

This example demonstrates creating text elements with Devize.

Code

const text = buildViz({
  type: "text",
  x: 150,
  y: 100,
  text: "Hello, Devize!",
  fontSize: "24px",
  fontFamily: "Arial",
  fill: "navy",
  textAnchor: "middle"
}, document.getElementById("visualization-container"));
