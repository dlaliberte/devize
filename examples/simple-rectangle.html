Simple Rectangle Visualization

This example demonstrates the most basic visualization in Devize - a colored rectangle.

Code

// Create a simple rectangle visualization
const rectangleViz = buildViz({
  type: "rectangle",
  x: 50,
  y: 50,
  width: 200,
  height: 100,
  fill: "steelblue",
  stroke: "black",
  strokeWidth: 2
}, document.getElementById("visualization-container"));
