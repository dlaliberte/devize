Circle Visualization

This example demonstrates creating a circle with Devize.

Code

const circle = buildViz({
  type: "circle",
  cx: 150,    // Center x-coordinate
  cy: 100,    // Center y-coordinate
  r: 50,      // Radius
  fill: "lightgreen",
  stroke: "darkgreen",
  strokeWidth: 2
}, document.getElementById("visualization-container"));
