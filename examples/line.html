Line Visualization

This example demonstrates creating a line with Devize.

Code

const line = createViz({
  type: "line",
  x1: 50,     // Start x-coordinate
  y1: 50,     // Start y-coordinate
  x2: 250,    // End x-coordinate
  y2: 150,    // End y-coordinate
  stroke: "purple",
  strokeWidth: 3
}, document.getElementById("visualization-container"));
