Text Visualization

This example demonstrates creating text with Devize.

Code

const text = buildViz({
  type: "text",
  x: 150,
  y: 100,
  text: "Hello, Devize!",
  fontSize: 24,
  fontFamily: "Arial",
  fill: "navy",
  textAnchor: "middle"  // Center-align text
}, document.getElementById("visualization-container"));
