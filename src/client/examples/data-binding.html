Advanced Data Binding

This example demonstrates Devize's declarative data binding capabilities, including field specifications and data transformations.

Data Transformations

Code

// Sample data
const salesData = [
  { product: "Product A", revenue: 420, profit: 210, category: "Electronics" },
  { product: "Product B", revenue: 650, profit: 300, category: "Clothing" },
  { product: "Product C", revenue: 340, profit: -50, category: "Electronics" },
  { product: "Product D", revenue: 570, profit: 140, category: "Food" },
  { product: "Product E", revenue: 320, profit: 90, category: "Clothing" }
];

// Create a bar chart with field specifications
const barChart = createViz({
  type: "barChart",
  data: salesData,
  x: { field: "product", type: "ordinal" },
  y: { field: "revenue", type: "quantitative" },
  color: { field: "category", type: "categorical" },
  tooltip: { fields: ["product", "revenue", "profit", "category"] }
}, document.getElementById("visualization-container"));

Filtering Data

// Filter data to show only high-revenue products
const filteredViz = createViz({
  type: "barChart",
  data: salesData,
  transforms: [
    { type: "filter", test: "d => d.revenue > 400" }
  ],
  x: { field: "product", type: "ordinal" },
  y: { field: "revenue", type: "quantitative" },
  color: { field: "category", type: "categorical" }
}, document.getElementById("visualization-container"));

Sorting Data

// Sort data by revenue
const sortedViz = createViz({
  type: "barChart",
  data: salesData,
  transforms: [
    { type: "sort", field: "revenue", order: "descending" }
  ],
  x: { field: "product", type: "ordinal" },
  y: { field: "revenue", type: "quantitative" },
  color: { field: "category", type: "categorical" }
}, document.getElementById("visualization-container"));
