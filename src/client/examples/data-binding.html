Data Binding Example

This example demonstrates how to bind data to visual elements in Devize.

Controls

Code

// Sample data
const data = [
  { product: "Product A", revenue: 420 },
  { product: "Product B", revenue: 650 },
  { product: "Product C", revenue: 340 },
  { product: "Product D", revenue: 570 },
  { product: "Product E", revenue: 230 }
];

// Create a bar chart with data binding
const barChart = createViz({
  type: "barChart",
  data: data,
  x: { field: "product" },
  y: { field: "revenue" },
  color: { field: "product" },
  tooltip: true
}, document.getElementById("visualization-container"));
